Tennis Volley


This is your chance to go to the next level.... by making a sprite game with multiple flying objects. The tennis balls need to go into an array, where you can move them and check any collisions using a forEach loop.

You can "mod" this game in any way you like - instead of tennis, go dodge flying cupcakes or fireballs!
To see a working example click here


Directions
  1. If you are coding outside of WoofJS.com (like in Sublime), insert the Woof_JS source tag into the <head>section of your project page:

    <script src="https://rawgit.com/stevekrouse/WoofJS/master/dist/woof.js"></script>

    Your code now speaks Woof!

  2. Select a background appropriate for your game. Create a Player with an image you like.

  3. Have the Player "glide" using direction keys. This means that when you press a direction key, you don't change the Player's x or y position but rather their xVelocity or yVelocity.

    HINT: don't forget to add friction to slow them down a little bit over time.

  4. Make sure the Player reacts properly when they hit an edge of the screen. They can pass thru to the other side of the screen, stop or bounce.

  5. Add a tennis ball object. Have one created Every second or two at the side of the screen. Remember to push each one into an array.

  6. Now, use a forEach loop to move each tennis ball. If it hits the edge of the screen, have it be deleted from the screen and from the array.

  7. Check if a tennis ball is hitting the Player. If so, remove it. Then add or subtract points, show a fun collision graphic and play a sound if you want. You can also increase tennis ball speed so the game gets harder.

  8. Add a score indicator on top of the screen.

  9. Now add your Opponent. Have the Opponent move in a random way and have the new tennis balls start at his position.

  10. If you want to have each Hit Animations look different, find different images and put them all into an Image Array. Then when you are creating a new Hit Animation, generate a random number, and assign the url: attribute to be one of the images in the Image Array, like: 
    
var hitAnimation = new Image ({
     url: arrayOfImages[randomNumber],
     height:   etc 
})

  11. Test and finish the project till it looks perfect to you.
